# 📊 Trader Performance vs Market Sentiment Analysis

## 📌 Project Overview

This project analyzes the relationship between **Bitcoin Market Sentiment (Fear & Greed Index)** and **Hyperliquid trader performance**. The objective is to understand how market sentiment influences trading behavior, profitability, and trading strategies.

This project was completed as part of the **Data Science / Data Analytics Intern – Round 0 Assignment** for **Primetrade.ai**.

---

# 🎯 Objective

The primary goals of this analysis are to:

* Analyze trader performance during **Fear** and **Greed** market conditions.
* Measure changes in trading behavior based on market sentiment.
* Identify different trader segments.
* Generate actionable trading recommendations backed by data.

# 📊 Key Metrics

The following metrics were analyzed:

* Daily Profit & Loss (PnL)
* Win Rate
* Average Trade Size
* Trade Frequency
* Long vs Short Ratio
* Position Size Distribution
* Trader Segmentation

> **Note:** The provided historical trading dataset did not include a **Leverage** column. Therefore, leverage-based analysis could not be performed. Position Size (Size USD) was used as an alternative indicator of trading aggressiveness.

# 🔍 Key Insights

The analysis focuses on identifying:

* Differences in trader profitability between Fear and Greed market conditions.
